# Covid-19-X-Ray-Recognition

### The objective of the project is to detect whether any person is infected with Covid-19 (Novel Coronavirus) or not.

It uses the Image classification of X-Rays for detection of the said virus.

The model is trained on the patients' and unaffected people's X-rays of lungs.

**NOTE:**
*The methods and techniques used in this project are not meant for a scientifically rigorous study. This article is for readers who are interested in (1) Computer Vision/Deep Learning and want to learn via practical, hands-on methods and (2) are inspired by current events. I kindly ask that you treat it as such.*
